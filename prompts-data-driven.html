10 Prompts Lipoxycut — Data-Driven

Chaque prompt génère l'image de fond uniquement. Le produit @img1 (packshot Lipoxycut) sera intégré via Canva/Photoshop en post-production, ainsi que les headlines. Les prompts laissent un espace libre pour le texte et le produit.

Workflow : Nano Banana génère l'image → Tu places @img1 (packshot) via Canva/Photoshop sans déformation → Tu ajoutes le headline + CTA par-dessus. Le prompt ne génère PAS le produit. Il génère le décor, la scène, la personne.
Copier = prompt dans le presse-papier. Le produit et le texte seront ajoutés via Canva/Photoshop.
